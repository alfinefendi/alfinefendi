Halo, senang dapat menulis kembali

Ada beberapa hal yang ingin saya ungkapkan, ini terkait topik dan rencana bagaimana saya ingin memulai karir sebagai seorang blogger / penulis di internet.

Saya berpikir, pada awalnya web ini hanya saya gunakan untuk placeholder atau static page apabila nanti telah masuk tes wawancara atau interview kerja wkwk. Maka saya kira saya bisa mengembangkan web ini bersamaan dengan niat saya tadi, ke arah dan tujuan yang lebih jelas seperti web informasi, web edukasi, web dokumentasi dan semacamnya. Dengan hal ini saya memiliki alasan yang kuat untuk tetap belajar.

Alih - alih web ini hanya memuat catatan hari - hari, lebih baik lagi apabila web ini memiliki peran dan fungsi. Seperti artikel seputar berkebun, berwirausaha. cara menanam, memilih bibit, dan seterusnya. lagi pula waktu saya sangat fleksibel sekali.

Sekian, sampai jumpa di lain waktu.
